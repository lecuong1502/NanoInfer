# Contributing to NanoInfer

First off, thank you for considering contributing to NanoInfer! 

## How to Contribute

1. **Fork** the repository and clone it locally.
2. **Create a branch** for your edits (`git checkout -b feature/your-feature-name`).
3. **Write code** and ensure it aligns with the project's C++/CUDA style.
4. **Test** your changes locally:
   ```bash
   make -j8
   pytest tests/ -v
